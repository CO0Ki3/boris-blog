## 성능 최적화 실습 과제

TGC UIX 기반 개발 이해 및 활용 방안 최적화 세션 실습 프로젝트 입니다.

### 실행

1. git clone

```
$ git clone https://github.com/koreacat/optimization-project.git
```

2. install dependencies

```
$ npm install
```

3. 프론트 개발 환경 서버 실행 

```
$ npm run start
```

4. api 서버 실행

```
$ npm run server
```

5. 배포 환경 서버 실행

```
$ npm run serve
```
