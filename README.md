## 성능 최적화 실습 과제

TGC UIX 기반 개발 이해 및 활용 방안 최적화 세션 실습 프로젝트 입니다.

### 실행

1. git clone

```
$ git clone https://github.com/koreacat/optimization-project.git
```

2. install dependencies

```
$ npm install
```

3. start development server

```
$ npm start
```

4. start server

```
$ npm run server
```
*3번의 dev server 와 다른 콘솔에서 띄워줍니다.
